Introduction

The ffmpeg_wrapper project is a lightweight C++ wrapper around FFmpeg 4.3 that simplifies the process of encoding, decoding, and manipulating multimedia files.

Project Dependencies

  • FFmpeg 4.3: The wrapper relies on FFmpeg 4.3 for multimedia processing. Please ensure you have FFmpeg installed and configured correctly.
  • Google Test (gtest): If you wish to run the tests, you need to have Google Test installed. You can install it via your package manager or build it from source.
